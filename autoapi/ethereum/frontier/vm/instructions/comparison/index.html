less_than¶

less_than(evm: ethereum.frontier.vm.Evm) → None¶

Checks if the top element is less than the next top element. Pushes the result back on the stack.

Parameters

evm – The current EVM frame.

Raises

  • StackUnderflowError – If len(stack) is less than 2.

  • OutOfGasError – If evm.gas_left is less than GAS_VERY_LOW.

def less_than(evm: Evm) -> None:
    evm.gas_left = subtract_gas(evm.gas_left, GAS_VERY_LOW)

    left = pop(evm.stack)
    right = pop(evm.stack)
    result = U256(left < right)

    push(evm.stack, result)

    evm.pc += 1
