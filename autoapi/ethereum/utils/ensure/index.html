ensure(value: bool, exception_class: Callable[], Exception] = EnsureError) → None¶

Does nothing if value is truthy, otherwise raises the exception returned by exception_class.

Parameters

  • value – Value that should be true.

  • exception_class – Constructor for the exception to raise.

def ensure(
    value: bool, exception_class: Callable[[], Exception] = EnsureError
) -> None:
    if value:
        return
    raise exception_class()
