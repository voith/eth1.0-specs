ensure
~~~~~~


.. function:: ensure(value: bool, exception_class: Callable[[], Exception] = EnsureError) -> None
   :noindexentry:

   Does nothing if `value` is truthy, otherwise raises the exception returned
   by `exception_class`.

   :param value: Value that should be true.
   :param exception_class: Constructor for the exception to raise.
