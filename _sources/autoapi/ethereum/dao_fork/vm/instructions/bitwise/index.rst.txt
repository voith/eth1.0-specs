bitwise_and

.. function:: bitwise_and(evm: ethereum.dao_fork.vm.Evm) -> None
   :noindexentry:

   Bitwise AND operation of the top 2 elements of the stack. Pushes the
   result back on the stack.

   :param evm: The current EVM frame.

   :raises ~ethereum.dao_fork.vm.error.StackUnderflowError: If `len(stack)` is less than `2`.
   :raises ~ethereum.dao_fork.vm.error.OutOfGasError: If `evm.gas_left` is less than `GAS_VERY_LOW`.

get_byte
~~~~~~~~


.. function:: get_byte(evm: ethereum.dao_fork.vm.Evm) -> None
   :noindexentry:

   For a word (defined by next top element of the stack), retrieve the
   Nth byte (0-indexed and defined by top element of stack) from the
   left (most significant) to right (least significant).

   :param evm: The current EVM frame.

   :raises ~ethereum.dao_fork.vm.error.StackUnderflowError: If `len(stack)` is less than `2`.
   :raises ~ethereum.dao_fork.vm.error.OutOfGasError: If `evm.gas_left` is less than `GAS_VERY_LOW`.
