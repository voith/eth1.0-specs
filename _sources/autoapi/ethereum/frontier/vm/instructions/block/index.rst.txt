block_hash
~~~~~~~~~~


.. function:: block_hash(evm: ethereum.frontier.vm.Evm) -> None
   :noindexentry:

   Push the hash of one of the 256 most recent complete blocks onto the
   stack. The block number to hash is present at the top of the stack.

   :param evm: The current EVM frame.

   :raises ~ethereum.frontier.vm.error.StackUnderflowError: If `len(stack)` is less than `1`.
   :raises ~ethereum.frontier.vm.error.OutOfGasError: If `evm.gas_left` is less than `20`.

coinbase
~~~~~~~~


.. function:: coinbase(evm: ethereum.frontier.vm.Evm) -> None
   :noindexentry:

   Push the current block's beneficiary address (address of the block miner)
   onto the stack.

   Here the current block refers to the block in which the currently
   executing transaction/call resides.

   :param evm: The current EVM frame.

   :raises ~ethereum.frontier.vm.error.StackOverflowError: If `len(stack)` is equal to `1024`.
   :raises ~ethereum.frontier.vm.error.OutOfGasError: If `evm.gas_left` is less than `2`.
