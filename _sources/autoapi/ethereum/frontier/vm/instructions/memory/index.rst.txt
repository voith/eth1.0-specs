mstore
~~~~~~


.. function:: mstore(evm: ethereum.frontier.vm.Evm) -> None
   :noindexentry:

   Stores a word to memory.
   This also expands the memory, if the memory is
   insufficient to store the word.

   :param evm: The current EVM frame.

   :raises ~ethereum.frontier.vm.error.StackUnderflowError: If `len(stack)` is less than `2`.
   :raises ~ethereum.frontier.vm.error.OutOfGasError: If `evm.gas_left` is less than
       `3` + gas needed to extend memeory.
